# K-Theme for webtrees 2.x

Modern, premium UI theme module with light and dark palettes. Upgrade-safe (no core modifications).

## Installation

1. Copy the `k-theme` folder into `webtrees/modules_v4/`.
2. In the webtrees control panel, enable **K-Theme** in **Modules â†’ Themes**.
3. Set the theme for your tree or as the default theme.
4. Open the module wrench icon and choose the **Light** or **Dark** palette.

## Supported versions

- webtrees 2.x

## Light/Dark usage

K-Theme provides two palettes: **Light** and **Dark**. Switch palettes using the webtrees theme palette selector.

## Notes

- No core files are modified.
- Assets live in `resources/`.
